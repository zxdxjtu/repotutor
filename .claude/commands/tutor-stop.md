# 停止讲解

停止当前的代码讲解并保存进度，以便下次继续。

## 执行操作

1. 停止音频播放：
   ```
   使用 repotutor-audio-player MCP 的 stop 工具
   ```

2. 保存会话状态到 `.repotutor/session/state.json`：
   - 当前章节和小节
   - 当前检查点索引
   - 用户提问历史
   - 时间戳

3. 显示保存的进度信息：
   - 已完成的章节
   - 进度百分比
   - 下次可使用 `/tutor --resume` 继续

## 输出信息

```
[RepoTutor] 讲解已停止

当前进度已保存:
- 位置: 系统架构 > 模块关系
- 进度: 35%
- 已用时间: 12 分钟
- 提问次数: 3

下次可使用 /tutor --resume 继续讲解
```
