# 讲解状态

显示当前的讲解进度和状态信息。

## 执行操作

1. 读取会话状态
2. 获取播放器状态：
   ```
   使用 repotutor-audio-player MCP 的 get_status 工具
   ```
3. 计算并显示进度信息

## 显示内容

```
[RepoTutor] 讲解状态

当前位置:
  章节: 系统架构
  小节: 模块关系
  检查点: src/services/index.ts:15

进度:
  已完成: 3/10 章节 (30%)
  预计剩余: ~25 分钟

播放状态: 播放中 / 已暂停 / 已停止

本次会话:
  开始时间: 10:30
  已用时间: 12 分钟
  提问次数: 2

配置:
  语音: Kore
  语言: 中文
  讲解深度: 标准
```

## 快捷操作提示

根据当前状态提供操作建议：
- 如果播放中：可以使用 /tutor-pause 暂停
- 如果已暂停：可以使用 /tutor-continue 继续
- 如果已停止：可以使用 /tutor 重新开始或 /tutor --resume 恢复
