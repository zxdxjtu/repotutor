# 暂停讲解

暂停当前正在进行的代码讲解。

## 执行操作

1. 调用音频播放器暂停：
   ```
   使用 repotutor-audio-player MCP 的 pause 工具
   ```

2. 保存当前讲解位置到会话状态

3. 显示当前位置信息：
   - 当前章节
   - 当前小节
   - 进度百分比

4. 提示用户可以：
   - 直接输入问题进行提问
   - 输入 `/tutor-continue` 继续讲解
   - 输入 `/tutor-goto <模块>` 跳转到其他部分
   - 输入 `/tutor-stop` 停止讲解

## 用户交互

暂停后，用户可以直接输入问题。回答问题时：
- 基于当前展示的代码上下文
- 引用具体的代码行和函数名
- 提供清晰的解释

回答完问题后，提示用户可以继续讲解或提出更多问题。
