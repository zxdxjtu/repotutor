# 第 1 章: 项目概览与核心理念

欢迎来到 RepoTutor 代码讲解！我是你的 AI 讲解助手，接下来我将带你深入了解这个创新的代码学习工具。

## 什么是 RepoTutor？

RepoTutor 是一个 AI 驱动的代码仓库智能讲解系统。它的核心理念是：让代码讲解不再依赖文字阅读，而是通过**语音合成**、**IDE 导航**和**交互式教程**三位一体，创造沉浸式的代码学习体验。

## 核心价值

传统的代码学习方式存在三大痛点：

1. **学习曲线陡峭** - 新人需要花大量时间阅读文档和代码
2. **缺乏引导** - 不知道从哪里开始，哪些是核心逻辑
3. **效率低下** - 需要在文档、代码和 IDE 之间频繁切换

RepoTutor 通过智能化的方式解决这些问题：

✨ **智能语音合成** - 使用 Google Gemini TTS 生成流畅的中文讲解，解放你的双眼
✨ **IDE 智能导航** - 自动在编辑器中打开相关代码，精准定位到关键行
✨ **结构化教程** - 自动分析代码仓库，生成层次分明的讲解大纲

## 技术架构

现在让我们来看看项目的技术栈。RepoTutor 采用了现代化的技术选型：

**运行时环境**：
- Node.js >= 18.0.0 - 提供稳定的 JavaScript 运行环境
- TypeScript - 类型安全，提升开发体验

**核心协议**：
- MCP (Model Context Protocol) - 这是 Anthropic 推出的模型上下文协议，允许 AI 应用通过标准化接口调用外部工具

**三大 MCP 服务器**：
1. **TTS 服务器** - 负责文本到语音的转换
2. **IDE 控制服务器** - 负责在 VSCode、Cursor 或 Qoder 中打开文件
3. **音频播放服务器** - 负责在 macOS 上播放生成的音频

## 项目结构

整个项目采用 npm workspaces 管理，主要包含：

- `mcp-servers/` - 3 个独立的 MCP 服务器
- `scripts/` - 自动化配置脚本
- `lib/` - 共享库（预留扩展）

这种模块化设计使得每个服务器职责单一、易于维护和扩展。
